# Cursor

Cursor – это объект представляющий собой курсор мыши, единственно через который пользователь взаимодейтсвует с web-инсталляцией.

# Принципы

1. Хочется иметь возможность контролировать темпо-ритм курсора пользователя и ощущение пользователя этого темпо-ритма, засчёт:

- Контроля физики движения
- Контроля размера курсора в зависимости от скорости движения
- Иконку курсора в зависимости от зоны под курсором
